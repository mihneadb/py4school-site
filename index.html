Py4School

Program pentru familiarizarea cadrelor didactice cu Python

ROSEdu, împreună cu Facultatea de Automatică și Calculatoare din cadrul Universității Politehnica din București, organizează Py4School - un set de training-uri ce au ca scop acomodarea profesorilor cu limbajul Python, urmat de alcătuirea programei unui CDS pentru predarea noțiunilor învățate elevilor de liceu.

De ce Python?

La ora actuală, elevii din licee învață programare în Pascal sau C/C++. Limbajul Pascal nu se mai folosește în industrie, iar limbajele precum C sunt bune pentru performanța lor, însă prezintă mari dezavantaje în zonele de învățare și expresivitate. În plus, Python este unul dintre cele mai populare limbaje de programare în ziua de azi, ceea ce înseamnă că se găsesc multe exemple și biblioteci utile pe Internet.

Față de C, Python prezintă următoarele avantaje:

  • se scrie mai puțin cod
  • indentarea face parte din sintaxă, ceea ce duce la cod mai ușor de citit
  • programele Python seamănă foarte mult cu cele în pseudocod
  • gestiune automată a memoriei
  • documentație accesibilă, cu exemple
  • debugging ușor
  • testarea se face ușor cu ajutorul unei biblioteci standard
  • se pune accentul pe algoritm, în loc de detalii de sintaxă
  • include structuri de date ușor de folosit
  • permite programarea orientată pe obiecte într-un mod facil

Desfășurare

În primă fază, vom organiza șase ședințe de pregătire în perioada 9 noiembrie - 14 decembrie, în fiecare sâmbătă. Acestea vor începe printr-o discuție introductivă și explicații, urmate apoi de workshop, unde vom rezolva exerciții pentru a aprofunda conceptele predate. ROSEdu va oferi și suport online de curs, cu documentație și exerciții./

După aceasta, în semestrul al doilea, vom oferi suport în desfășurarea unei clase de Python ca CDS sau în cadrul cluburilor de informatică. Pentru aceasta vă vom pune la dispoziție un manual, site de curs, exemple de teme și, în plus, vom fi prezenți în cadrul ședințelor pentru a ajuta cu suport tehnic, dacă este necesar.

Echipa

Programatori Python cu experiență, studenți pasionați de limbaj și de programare.
